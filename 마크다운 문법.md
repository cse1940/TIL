# 마크다운 문법

## 1. Heading

- 문서의 제목이나 소제목으로 사용
  - #의 갯수에 따라 대응되는 수준이 있다.
  - h1 ~ h6



## 2. List

- 순서가 있는 리스트
- 순서가 없는 리스트



## 3. Fenced Code block

- 코드 블록은 (```아이폰11```)로 작성
- 코드 블록에 특정 언어를 명시하면 형광펜 적용 가능



## 4. Inline Code block

- 코드 블록은 (``)로 작성



## 5. Link, 이미지

- [문자열](url) 을 통해 특정 파일들 포함하여 연결시킬 수도 있음
- ![문자열](url)



## 6. 인용문

- (>) 를 통해 인용문 작성



## 7. table(표)

- 본문(P) -> 표



## 8. text 강조

- 굵게 (** **)
- 기울임 (* *)



## 9. 수평선

- (***), (---), (___) 



## 10. 저장

- ctrl + s
